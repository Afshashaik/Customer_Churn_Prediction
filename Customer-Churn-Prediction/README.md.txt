This project predicts customer churn using Machine Learning models.

## Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost

## Models Used
- Logistic Regression
- Random Forest
- XGBoost

## Evaluation Metrics
- Accuracy
- Recall
- ROC-AUC Score

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Churn Prediction
- Data Visualization
